# bashrc_file
My bashrc file

## Install Steps 

1.) create a bashrc file (backing up old one first)
```shell
$ cd ~
$ mv ~/.bashrc ~/.bashrc.bak   
$ vim ~/.bashrc
``` 

2.) 
copy contents of .bashrc from this repository

3.) paste in the contents of the copied file 
```
<esc> :a 
<cmd-v>
``` 

4.) save file
```
<esc> :wq
``` 

4.) update shell .bashrc source
```shell
$ source .bashrc 
```
